# Mobile — NFC

## Абстракция ридера
[`src/nfc/index.ts`](../../../apps/mobile/src/nfc/index.ts) — обёртка `nfcReader` над
`react-native-nfc-manager`:
- `isAvailable()` — поддерживается ли NFC на устройстве;
- `readUid()` — считывает UID метки (NfcTech.NfcA → `tag.id`, нормализуется в нижний регистр hex).

Всё чтение обёрнуто в try/catch + `finally` с отменой технологического запроса — NFC может
быть выключен или не поддерживаться. Требуется **физическое устройство** + dev-build (в Expo Go
нативный модуль недоступен).

## Сопоставление UID → точка маршрута
На экране обхода ([`app/patrol.tsx`](../../../apps/mobile/app/patrol.tsx), компонент `ScanAction`):
1. `readUid()` → нормализованный UID.
2. Поиск точки: `points.find(p => p.nfcTag?.uid === uid)`.
3. Валидация на клиенте:
   - метка не из этого маршрута → ошибка;
   - точка уже отмечена → ошибка;
   - метка не соответствует текущей ожидаемой точке → экран показывает подсказку, какую точку отметить, событие скана не создается, а попытка пропуска отправляется в `POST /api/v1/mobile/patrols/:id/missed-point-attempts`.
4. Успех → `createLocalEvent()` (см. [offline-sync.md](offline-sync.md)).

Попытка пропуска нужна, чтобы менеджер видел нарушение даже при строгом порядке маршрута на клиенте. Backend создает инцидент `missed_point`, если сотрудник попытался отсканировать точку с большим `sortOrder`, чем текущая ожидаемая точка.

## UX сканирования

Перед чтением NFC приложение проверяет:

- поддерживается ли NFC на устройстве;
- включен ли NFC в настройках телефона.

Если NFC выключен, показывается стилизованный диалог с предложением открыть настройки. Во время чтения показывается затемненный NFC overlay; пользователь видит, что приложение находится в режиме сканирования.

## Настройка маршрута и привязка меток
- `route-setup/` — admin/manager задаёт число точек и поочерёдно привязывает NFC-метки
  (`mobile/shops/:shopId/route-setup/*`).
- `nfc-replace/` — замена метки у существующей точки (`patrol-points/:id/replace-nfc`).

Конфиг плагина и разрешение NFC — в `app.json`
(`react-native-nfc-manager` + permission `NFC`).
